# laravel vue todo list
### how to install and use:
```
git clone https://github.com/yasinsvr/laravue-todo.git
cd laravue-todo
cd backend
composer install
php artisan serve

cd ../frontend
npm install
npm run dev
```
## go to this page
```
http://localhost:5173/
```
