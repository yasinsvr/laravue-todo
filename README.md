# laravel vue todo list
### how to install and use:
```
git clone https://github.com/yasinsvr/laravue-todo.git
cd laravue-todo
cd backend
composer install
php artisan migrate
php artisan serve
```
after installed and run backend open new terminal in laravue-todo folder and run this commands:
```
cd frontend
npm install
npm run dev
```
